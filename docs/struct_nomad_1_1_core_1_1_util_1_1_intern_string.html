Detailed Description

Represents an interned string, providing efficient storage and comparison of string values.

An InternString is a lightweight value type that stores an integer identifier pointing to a unique string instance in the global string pool. This enables:

  • Reduced memory usage for duplicate strings
  • Fast equality comparisons (integer comparison instead of string comparison)
  • Efficient hashing (the identifier itself serves as the hash code)

Interned strings are managed by the StringPool class, which ensures that identical strings share the same storage and identifier.

This struct implements IEquatable<T> for efficient equality comparisons without boxing. See the remarks on the Equals(InternString) method for details.

◆ GetHashCode()

override int Nomad.Core.Util.InternString.GetHashCode ( )
inline

Returns the hash code for this instance.

Returns
The identifier of this interned string, which serves as its hash code.

The hash code is simply the integer identifier, which provides:

  • Perfect distribution (each interned string has a unique identifier)
  • Consistency with equality (equal instances have the same hash code)
  • Minimal computation (the identifier is already available)

This method is aggressively inlined for performance.

Converts an InternString to a regular string.

Parameters
valueThe InternString to convert.
Returns
The string value represented by the InternString, or string.Empty if the identifier is invalid or represents an empty string.

This conversion looks up the string value in the string pool using the identifier. If the identifier is invalid (does not correspond to an interned string), string.Empty is returned.

This method is aggressively inlined for performance, though the underlying string pool lookup may involve a dictionary lookup.
