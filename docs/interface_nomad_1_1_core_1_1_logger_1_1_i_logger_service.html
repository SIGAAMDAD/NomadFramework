void Nomad.Core.Logger.ILoggerService.AddSink ( ILoggerSink sink)

Adds a sink to the logger.

A sink is a dedicated output abstraction, that prints messages to IO outputs such as stdout, a logging file, or the in-game console. You can setup your own sink if you want to route log messages elswhere.

Parameters
sinkThe sink to add.

ILoggerCategory Nomad.Core.Logger.ILoggerService.CreateCategory ( string name,
LogLevel level,
bool enabled )

Creates a new logging category.

A logging category is a denotation of where a message is coming from for easier debugging and tracing. You can also filter categories by enabling or disabling them as well as by LogLevel.

Parameters
nameThe name of the category.
levelThe log level of the category.
enabledWhether the category is enabled. If false, any log messages sent with this category as a parameter will not send any messages to the sinks.
Returns
The created logging category.
