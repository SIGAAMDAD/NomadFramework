◆ EventExceptionPolicy

enum Nomad.Core.Events.EventExceptionPolicy
Enumerator
Propagate 

First subscriber exception immediately aborts publishing. Best for command-style events where failure means the operation is invalid.

AggregateAfterDispatch 

Run every subscriber, collect all failures, then throw one aggregate exception. Best default for editor/debug/framework-level correctness.

ReportAndContinue 

Run every subscriber, log/report failures, never throw from Publish. Best for hot-path runtime events like input, UI notifications, telemetry, audio state changes.
