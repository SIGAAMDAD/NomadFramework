Detailed Description

A typed generic CVar interface. Used for referencing CVars in code as seen in the examples provided below.

A CVar's type can only be one of the following:

  • Int32 A 32 bit signed integer value.
  • UInt32 A 32 bit unsigned integer value.
  • Boolean An 8 bit boolean true-false value.
  • Decimal A 32 bit floating-point value.
  • String A null-terminated string value.
