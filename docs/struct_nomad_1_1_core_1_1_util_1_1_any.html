Detailed Description

A type-safe container that can hold any of the supported primitive types. Similar to C++'s std::any, it provides a unified way to store and retrieve different value types in a memory-efficient manner using a union structure.

The Any struct occupies exactly 20 bytes of memory regardless of the stored type, using an explicit layout union to share the same memory space.

This type is particularly useful in scenarios where type erasure is needed while maintaining type safety through explicit conversion or retrieval.
