The metadata of a persistent game state file. Stores relevant information for debugging, listing, and/or sorting.

Retrieving the metadata for all the save files currently available can be done with Services.ISaveDataProvider.ListSaveFiles.

Parameters
SaveNameThe save file's name. This is not the actual name, just the user-provided identification.
FileSizeThe size of the save file in bytes.
LastAccessYearThe year the save file was last accessed.
LastAccessMonthThe month the save file was last accessed.
LastAccessDayThe day the save file was last accessed.
