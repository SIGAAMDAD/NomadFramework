 CBankHandleRepresents a handle to an audio bank within the system. This struct encapsulates a 32-bit unsigned integer value that serves as a reference to a loaded audio bank, allowing for efficient management and access to audio resources
 CChannelGroupHandleA handle that uniquely identifies a channel group within the audio system. This struct is used to manage and reference channel groups for audio routing and processing
 CChannelHandleRepresents a handle to an audio channel within the Nomad audio system. This struct encapsulates a 32-bit integer value that serves as a reference to an active audio channel, allowing for efficient management and control of audio playback. The ChannelHandle struct provides type safety and abstraction over raw integer values, ensuring that operations on audio channels are performed correctly and safely. By implementing the IValueObject<T> interface, it also supports value-based equality comparisons, making it easier to manage collections of channel handles and perform operations such as lookups and comparisons within the audio system
 CEventHandleA handle that uniquely identifies an audio event instance. This struct is used to track and manage active audio events within the system
