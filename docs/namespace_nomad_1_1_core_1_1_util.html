enum Nomad.Core.Util.AnyType : byte

Represents the type of value stored in an Any instance.

This enumeration is used internally by the Any struct to track the type of value stored, enabling type-safe operations and conversions.

Enumerator
None 

No type, default construction.

Boolean 

Represents a boolean value.

Int8 

Represents a signed 8-bit integer.

Int16 

Represents a signed 16-bit integer.

Int32 

Represents a signed 32-bit integer.

Int64 

Represents a signed 64-bit integer.

UInt8 

Represents an unsigned 8-bit integer.

UInt16 

Represents an unsigned 16-bit integer.

UInt32 

Represents an unsigned 32-bit integer.

UInt64 

Represents an unsigned 64-bit integer.

Float32 

Represents a 32-bit floating-point number.

Float64 

Represents a 64-bit floating-point number.

String 

Represents a string.

Count 

Represents the total count of valid AnyType values.

This value is useful for array allocations and bounds checking. It is not a valid type for storing values in an Any instance.
